# 项目简介

## 核心功能
- 数据采集：自动爬取豆瓣电影数据，收集电影相关信息。
- 数据清洗：对收集到的数据进行处理，确保数据质量和准确性。
- 数据分析：对清洗后的数据进行深入分析，挖掘电影产业趋势。
- 可视化展示：利用Echarts等工具，将分析结果以图表形式直观展示。

## 适用场景
- 电影行业研究：为电影研究者提供数据支持，帮助他们分析电影市场趋势。
- 影迷社区：为影迷提供深入了解电影产业的工具，帮助他们发现和了解更多电影。
- 媒体机构：为媒体机构提供数据支持，帮助他们进行电影报道和分析。

## 技术特点
- Python技术栈：采用Python作为主要开发语言，实现高效的数据处理和分析。
- Echarts图表库：利用Echarts强大的图表功能，实现数据可视化。
- Flask框架：采用Flask框架构建Web应用，实现数据展示和交互。
- MySQL数据库：存储和处理大量数据，确保数据的安全性和可靠性。
